# 🎨 Color Picker

A simple web application that allows users to change the background color of the page by clicking different color buttons.

## 🚀 Live Demo
[Click Here to View Project](https://lingappagarivinod.github.io/Color-Picker/)

---

## 📌 Features

- Change background color dynamically
- Displays selected hex color code
- Clean and responsive UI
- Built using pure HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
